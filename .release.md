## 一键安装 / 升级

首次安装：
```bash
# 挑一个干净的目录（强烈建议不要在 $HOME 或 / 下跑）
mkdir -p ~/blog && cd ~/blog
curl -fsSL https://github.com/LJW0401/blog-server/releases/download/<VERSION>/manage.sh -o manage.sh
chmod +x manage.sh
sudo ./manage.sh install
```

国内网络加速（走镜像）：
```bash
sudo GH_MIRROR=https://ghproxy.com/ ./manage.sh install
```

已有部署升级：
```bash
cd /path/to/your/install
sudo ./manage.sh update
```

日常运维：
```bash
sudo ./manage.sh restart
sudo ./manage.sh logs           # 实时
sudo ./manage.sh logs 200       # 最近 200 行
sudo ./manage.sh status
```

## 附件说明 📦

| 文件 | 说明 |
|------|------|
| `manage.sh` | 一键安装 / 升级 / 运维脚本，上面所有命令都靠它 |
| `blog-server-linux-amd64.tar.gz` | Linux x86_64 打包产物（含二进制 + service unit），离线部署可用 |
| `blog-server-linux-amd64.tar.gz.sha256` | 上面 tarball 的 sha256，跨机传输做完整性校验 |
| `blog-server` | 裸二进制（Linux x86_64，CGO_ENABLED=0），自己写 systemd 时可直接拿 |
| `blog-server.sha256` | 裸二进制的 sha256 |
| `blog-server.service` | 参考用 systemd unit 文件，`manage.sh install` 会自动铺好，自定义部署时可拿来改 |

> 日常 install / update 只需要 `manage.sh`，脚本会自动拉取同版本的其它附件并校验 sha256。
